# Side Project Team Matching Service(서비스명 미정)
### 사이드 프로젝트를 같이 진행할 팀원을 모집하는 서비스

---
#### DESCRIPTION
* 포트폴리오 혹은 재미를 위한 사이드 프로젝트를 함께 진행할 팀원을 구하는 플랫폼 서비스
* 사용자는 프로젝트를 등록하여 같이 진행할 팀원을 모집하거나 올라온 프로젝트에 참여 요청을 보낼 수 있음
